How to post updates

Add a new object to updates.json and write the notes in Markdown. That becomes the detailed release note users see here.

  • Use title for the headline users should see.
  • Use summary for the short preview text.
  • Use notes for detailed release notes in Markdown.
  • New entries appear automatically on this page.
